Drama Codex OAuth Flow

Drama does not store OpenAI API keys or OAuth tokens. The Electron UI starts the PlotPilot sidecar, calls PlotPilot LLM Control, and delegates ChatGPT login plus generation to the official Codex app-server over stdio JSON-RPC.

Drama Responsibilities

  • - Render PLM login/status UI
  • - Start PlotPilot sidecar
  • - Open returned auth_url
  • - Persist graph/project outputs

PlotPilot Responsibilities

  • - Expose llm-control endpoints
  • - Select protocol=codex provider
  • - Translate generation into JSON-RPC
  • - Stream chapter text back

Security Boundary

  • - OAuth tokens remain in Codex CLI
  • - No OpenAI API key required
  • - Drama only sees status/auth URL
  • - Craft runtime path is separate
Generated from Drama commit c69d117 and PlotPilot v4.6 Codex app-server adapter source.
